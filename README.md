# South East Watersports Website

A premium 3-page static HTML website for Epic Life's watersports landing page, designed to convert Google Ads visitors into bookings.

## 🎨 Brand

- **Company:** Epic Life (trading as South East Watersports)
- **Logo:** Included from Epic Life CDN
- **Primary Color:** Sky Blue rgb(108, 197, 237)
- **Dark Accent:** Navy #0a1628
- **Font:** Inter (Google Fonts)
- **Style:** Modern, clean, outdoorsy adventure brand

## 📄 Pages

1. **index.html** - Homepage with hero, why choose us, locations, testimonials
2. **paddleboarding.html** - All SUP sessions (Taster, Mega SUP, Social Paddle, SUP Yoga, Intermediate)
3. **kayaking.html** - Kayaking sessions (Taster & Adventure, Sea Kayaking)

## ✨ Features

- **Fully Responsive** - Mobile-first design with hamburger menu
- **Smooth Animations** - IntersectionObserver scroll animations
- **Sticky Navbar** - Transparent on load, solid on scroll
- **FareHarbor Integration** - Booking calendar embedded
- **Premium Design** - Card-based layouts with hover effects
- **High-Quality Images** - Curated Unsplash photography
- **Conversion-Focused** - Multiple CTAs, clear booking paths

## 🔧 Technical

- Semantic HTML5
- CSS Grid & Flexbox
- Vanilla JavaScript (no dependencies)
- Google Fonts (Inter)
- FareHarbor lightframe script

## 🚀 Deployment

Open `index.html` in a browser or deploy to any static hosting service (Netlify, Vercel, GitHub Pages, etc.).

All assets are externally hosted (logos, images), so no build process required.
